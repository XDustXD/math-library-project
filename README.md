# MathLibraryProject

Простое .NET‑решение, состоящее из библиотеки математических функций и консольного приложения для её демонстрации.

## Обзор

- **MathLibrary** – библиотека классов (`MathLibrary.dll`) с единственным статическим классом `Calculator`, предоставляющим базовые арифметические операции и несколько дополнительных методов.
- **MathLibrary.Client** – консольное приложение, ссылающееся на библиотеку и выводящее примеры работы её методов.

## Возможности библиотеки

Класс `Calculator` (файл `MathLibrary/Calculator.cs`) реализует следующие методы:

- `Add(double a, double b)` – сложение двух чисел.
- `Subtract(double a, double b)` – вычитание второго числа из первого.
- `Multiply(double a, double b)` – умножение.
- `Divide(double a, double b)` – деление; при `b == 0` выбрасывает `DivideByZeroException`.
- `IsPrime(int number)` – проверка целого числа на простоту.
- `Power(double number, double power)` – возведение в целую степень (поддерживаются положительные и отрицательные показатели).
- `Factorial(int n)` – факториал неотрицательного целого числа (для отрицательного значения выбрасывается исключение).
- `SolveQuadratic(double a, double b, double c, out double? x1, out double? x2)` – решение квадратного уравнения. Возвращает `true` и корни при наличии действительных корней, иначе `false` и `null` в выходных параметрах.

Все методы статические и снабжены XML‑комментариями.

## Консольное приложение

Программа `MathLibrary.Client/Program.cs` демонстрирует:

1. Базовые арифметические операции.
2. Обработку исключения при делении на ноль.
3. Проверку массива чисел на простоту.
4. Вычисления степеней и факториалов.
5. Решение квадратных уравнений на примерах.

Результаты выводятся в консоль.

## Сборка и запуск

Решение компилируется с помощью .NET SDK (целевая платформа .NET 10.0).

Выполните из корня репозитория:

```bash
# собрать решение
dotnet build Mathclasslib.sln

# запустить клиент
dotnet run --project MathLibrary.Client/MathLibrary.Client.csproj
```

Сгенерированный файл `MathLibrary.dll` может быть подключён к другим .NET‑проектам.

## Расширение библиотеки

Чтобы добавить новую функцию:

1. Добавьте статический метод в класс `Calculator` с соответствующей XML‑документацией.
2. При наличии — добавьте модульные тесты или расширьте демонстрацию в клиенте.
3. Пересоберите решение.

## Примечания

- Метод `Power` выполняет повторяющееся умножение и не поддерживает дробные показатели.
- Для отрицательных степеней возвращается обратное значение положительной степени.
- `SolveQuadratic` возвращает `false`, если действительных корней нет; выходные параметры `x1` и `x2` устанавливаются в `null`.
